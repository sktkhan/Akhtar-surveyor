QUICK REFERENCE

Formulas & References

Whole Circle Bearing

WCB = atan2(ΔE, ΔN) × (180/π)

If WCB < 0, add 360°. Result: 0° to 360°

Reduced Level (HI Method)

HI = BM + BS
RL = HI − FS

Height of Instrument = Bench Mark + Back Sight

Circular Curve

T = R·tan(Δ/2)
L = R·Δ (radians)
E = R·(sec(Δ/2) − 1)

T=Tangent, L=Arc Length, E=External Distance

Area by Coordinate (Shoelace)

A = ½ |Σ(Eᵢ·Nᵢ₊₁ − Eᵢ₊₁·Nᵢ)|

Works for any polygon with n ≥ 3 coordinate points

Traverse Accuracy

Ratio = Total Length / Misclosure

1st Order ≥ 1:10000 | 2nd Order ≥ 1:5000 | 3rd ≥ 1:2000

Vertical Curve

K = L / |G2 − G1|

K-value = Rate of change of grade. Sag if G2 > G1
